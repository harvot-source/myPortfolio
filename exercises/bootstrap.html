CREATING CONTAINERS WITH BOOTSTRAP

    Three different types of the container
  • Fixed - this is the .container class that provides a responsive fixed width container
  • Fluid - this is the .container-fluid that provides a full width container spanning the entire widht of the viewport.
  • Responsive containers - allows you to specify class that is 100% wide until the specified breakpoint is reached. The breakpoints can be sm, md, lg, lx, xxl.
